# Mesto

Сервис для добавления фотографий, лайков и редактирования профиля.



## Команды

```bash
# Установка зависимостей
npm install

# Запуск локального сервера разработки
npm run dev

# Сборка проекта
npm run build

# Публикация на GitHub Pages
npm run deploy
```

## Технологии

- HTML, CSS, JavaScript
- Vite
- REST API
