# Mesto

Сервис для добавления фотографий, лайков и редактирования профиля.

## Ссылка на опубликованный проект

https://kirill666-1.github.io/mesto-production/

## Команды

```bash
# Установка зависимостей
npm install

# Запуск локального сервера разработки
npm run dev

# Сборка проекта
npm run build

# Публикация на GitHub Pages
npm run deploy
```

## Технологии

- HTML, CSS, JavaScript
- Vite
- REST API
